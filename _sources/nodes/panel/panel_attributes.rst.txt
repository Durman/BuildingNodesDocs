=====================
Panel Attributes Node
=====================

This node outputs :ref:`panels` attributes as :ref:`fields`.

Outputs
-------

Index
  Index of the panel in a row (floor) of panels. It is incremented by fill
  socket of :doc:`/nodes/floor/floor`.
