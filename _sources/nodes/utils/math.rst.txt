=========
Math Node
=========

Node performs basic mathematical operations.

Inputs
------

Float
  First float value

Float
  Second float value

Properties
----------

Operation
  Mode which points to what operation to use

  * **add:** addition
  * **subtract:** subtraction
  * **multiply:** multiplication
  * **divide:** division
  * **reminder:** remainder of the division

Outputs
-------

Float
  Resulting value
