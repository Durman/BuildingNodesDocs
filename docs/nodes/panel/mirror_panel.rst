=================
Mirror Panel Node
=================

This node can mirror a panel along vertical or horizontal axis.

Inputs
------

Panel
  Panel to be mirrored

Properties
----------

Mirror along X
  This will mirror panel from left to right

Mirror along Y
  This will mirror panel upside down

Outputs
-------

Panel
  Mirrored panel
