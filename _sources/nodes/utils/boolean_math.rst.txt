=================
Boolean Math Node
=================

The node performs logical operations.

Inputs
------

Boolean value 1
  First operand

Boolean value 2
  Second operand

Properties
----------

Operation
  - **And**: If both values are true it returns true otherwise false
  - **Or**: If at least on value is tur it returns tre otherwise false
  - **Not**: Converts first value from true to false or from false to true

Outputs
-------

Boolean value
  Result of the operation
